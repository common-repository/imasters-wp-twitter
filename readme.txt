== Description ==

With iMasters WP Twitter you can add a Retweet button to yours WordPress posts and update your Twitter status when publish some post.

== Installation ==

How to install iMasters WP Twitter

e.g.

1. Upload `imasters-wp-twitter` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP Twitter' menu in WordPress
